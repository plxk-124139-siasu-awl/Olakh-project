# Olakh

Olakh is a prototype platform that enables local Indian businesses to quickly build and launch their own websites with minimal input, focusing on accessibility, speed, and simplicity.

## 🚀 What it does

Olakh allows small business owners to create a simple website in minutes without any technical knowledge.
Users can enter basic details and instantly get a functional business webpage.

## 🧠 Problem it solves

Many local businesses in India are not online due to lack of technical skills or resources.
Olakh simplifies this by providing a fast, no-code solution.

## ⚙️ Features

* Instant website generation
* Simple and clean UI
* Dashboard for managing websites
* WhatsApp integration concept
* Designed for local businesses

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## ▶️ How to run

1. Download or clone the repository
2. Open `index.html` in your browser
